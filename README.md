# Global News Market Impact

미국 주요 기술 기업의 뉴스가 기사 이후 첫 정규장 주가에 미치는 영향을 분석하는 프로젝트입니다.

현재는 구현 직전의 기획 단계입니다. 첫 MVP의 범위를 작게 유지하고, 결과가 유의미할 때만
대상 기업과 시장 변수를 확장합니다.

## 핵심 질문

> 기업 뉴스만 사용했을 때보다 기사 게시 전 시장 심리와 주요 정책·경제 이벤트를 함께
> 사용했을 때 기사 이후 첫 정규장의 주가 상승 여부를 더 정확히 예측할 수 있는가?

이 프로젝트가 예측하는 것은 개인 투자자의 실제 심리나 행동이 아닙니다. 과거 기사 이후의
실제 주가 움직임을 학습하여 새로운 기사의 상승 가능성을 추정합니다.

품질을 확인한 뒤 필요할 때만 추가합니다.

## MVP 입력과 출력

### 입력

- 기사 제목과 요약
- 관련 기업 티커
- 기사 최초 게시 시각
- 기사 게시 전 거래일의 공포·탐욕지수
- 최근 24시간 내 주요 정책·경제 이벤트 유무

### 출력

- 기사 이후 첫 정규장 종가 기준 `상승` 또는 `비상승`
- 상승 확률
- 판단에 크게 반영된 기사 표현 또는 변수

초기 상승 라벨은 기사 직전 확정 종가보다 기사 이후 첫 정규장 종가가 높은지를 기준으로
만듭니다. 이후 결과의
노이즈가 크면 나스닥 100 대비 초과수익률이나 최소 변동 폭을 적용합니다.

## 주요 정책 및 경제 이벤트

시장 전체에 영향을 주는 모든 소식을 수집하지 않고 다음 유형만 보조 정보로 사용합니다.

- 미국 대통령 및 행정부의 관세, 무역, 반도체, AI 규제 발표
- 연방준비제도의 기준금리 결정과 주요 통화정책 발표
- 소비자물가지수와 고용지표 등 주요 미국 경제지표 발표

대통령 관련 정보는 Truth Social 게시물과 백악관 공식 발표 중 시장 또는 대상 산업과 직접
관련된 내용만 후보로 사용합니다. MVP에서는 본문을 별도로 학습하지 않고 최근 24시간 내
중요 이벤트 발생 여부를 단순 플래그로 연결합니다.

## 필요한 데이터

### 기업 뉴스

```text
article_id, title, summary, published_at, source, url, ticker
```

- 최초 게시 시각과 수정 시각 구분
- 기업명과 티커 연결
- 동일 사건의 중복 기사 식별
- 기사 전문 대신 제목과 요약 중심으로 시작

### 종목 및 시장 가격

```text
ticker, trading_date, open, high, low, close, adjusted_close, volume
```

- 대상 8개 기업의 일봉
- 시장 비교용 QQQ 일봉
- 미국 증시 거래일과 휴장일

### 시장 심리

```text
date, fear_greed_score, fear_greed_category
```

공포·탐욕지수의 과거 데이터를 안정적으로 확보하기 어렵다면 VIX 하나로 대체합니다.
기사 게시 시점 이후에 확정된 값을 사용하지 않도록 기사 게시 전 거래일 값을 연결합니다.

### 정책 및 경제 이벤트

```text
event_id, event_type, published_at, source, title, importance
```

초기 이벤트 유형은 `TARIFF_POLICY`, `FOMC`, `CPI`, `EMPLOYMENT`로 제한합니다.

## 시간축 결합 규칙

기업 뉴스, 시장 전체 이벤트, 시장 심리, 가격 데이터는 날짜만이 아니라 정확한 게시·발표
시각을 기준으로 하나의 학습 행에 연결합니다. 모든 시각은 원본 시간대와 함께 저장하고,
분석 시 미국 동부시간으로 정규화합니다.

```text
기업 기사 최초 게시 시각
+ 게시 시각 이전 24시간의 주요 시장 이벤트
+ 게시 시점에 이미 확정된 시장 심리 값
+ 기사 직전 확정 종가와 기사 이후 첫 정규장 종가
-> 상승 또는 비상승 라벨
```

가격 데이터에는 기사 직전 거래일과 기사 이후 첫 정규장의 시가·종가·거래량이 필요합니다. 대상 8개
종목과 QQQ에 동일한 거래일 기준을 적용하고, 미국 증시 휴장일과 조기 폐장일도 반영합니다.

기사 게시 시점별 기본 처리 규칙은 다음과 같습니다.

- 장중 기사: 직전 거래일 종가와 당일 정규장 종가를 비교
- 장 마감 후 기사: 당일 정규장 종가와 다음 정규장 종가를 비교
- 주말·휴일 기사: 직전 개장일 종가와 다음 개장일 종가를 비교
- 상승 라벨: 기사 직전 확정 종가보다 기사 이후 첫 정규장 종가가 높으면 생성

이 규칙은 일봉 기반 MVP에서 장중·장외·주말 기사를 일관되게 처리하기 위한 초기안입니다.
장중 기사 반응을 더 세밀하게 측정할 필요가 확인되면 분봉 데이터를 사용하는 방식으로
확장합니다.

정책 및 경제 이벤트 역시 기사보다 먼저 발표된 정보만 연결합니다. 기사 이후 발표된 관세,
금리, CPI, 고용지표나 당일 종가로 확정되는 시장 심리 값을 입력에 포함하면 미래 정보가
섞이므로 사용할 수 없습니다.

## 예상 처리 흐름

```text
뉴스 데이터 확보 가능성 및 기사량 조사
-> 기사 수집과 중복 제거
-> 기사와 기업 티커 연결
-> 게시·발표 시각을 미국 동부시간으로 정규화
-> 게시 시점 이전 시장 심리와 최근 24시간 이벤트 연결
-> 기사 직전 확정 종가와 기사 이후 첫 정규장 종가 연결
-> 상승 또는 비상승 라벨 생성
-> 기준 모델 학습
-> 시장 심리와 이벤트 추가 모델 학습
-> 두 모델의 성능 비교
```

## 비교 실험

1. 기사 제목과 요약만 사용하는 기준 모델
2. 기사와 전일 공포·탐욕지수를 사용하는 모델
3. 기사, 전일 공포·탐욕지수, 주요 이벤트 플래그를 사용하는 모델

평가는 시간순으로 학습·검증 데이터를 나누고 Accuracy, Macro F1, 상승 확률의 신뢰도를
비교합니다. 기사 이후 정보가 입력에 포함되는 데이터 누출을 반드시 방지합니다.

## 시작 순서

1. 8개 기업의 최근 2년 뉴스 데이터 확보 가능성과 월별 기사 수 조사
2. 뉴스 공급자와 가격 데이터 공급자 확정
3. 최소 수집 파이프라인과 데이터 스키마 설계
4. 기사 직전 종가와 기사 이후 첫 정규장 종가를 연결한 학습 데이터 생성
5. 기사만 사용하는 기준 모델 구현
6. 시장 심리와 정책 이벤트 변수를 순서대로 추가

## 확장 조건

MVP 데이터 품질과 예측 결과가 확인된 뒤에만 다음 항목을 검토합니다.

- META, AMZN 등 대상 기업 추가
- WMT, COST 등 소비·유통 업종 추가
- 초과수익률 기반 3단계 신호
- VIX, 풋·콜 비율, 나스닥 모멘텀 등 시장 변수 추가
- 장중 및 장외 가격 데이터
- 실시간 예측과 증권사 API 연동

## 유의사항

이 프로젝트는 연구 및 정보 분석 목적입니다. 결과는 실제 수익이나 특정 종목의 매수를
보장하지 않으며 직접적인 투자 권유로 사용하지 않습니다.
